API em FastAPI que retorna dados de times da NBA e uma análise gerada por IA sobre o confronto entre dois times.
Instalação
1. Instalar Ollama

No PowerShell:

irm https://ollama.com/install.ps1 | iex

2. Baixar modelo

ollama run llama3:8b

3. Instalar dependências Python

pip install fastapi uvicorn nba_api pandas requests

Rodar o projeto

uvicorn main:app --reload

Endpoints
Jogos por data

GET /games?date=YYYY-MM-DD

Retorna os jogos de uma data específica com times, data, horário e logos.
Dados do confronto

GET /teams/{team_id}
GET /matchups/{team1_id}/{team2_id}/history
GET /matchups/{team1_id}/{team2_id}/top-scorers?limit=15
GET /matchups/{team1_id}/{team2_id}/top-players?limit=10

Em conjunto, esses endpoints retornam:

    informações dos times
    estatísticas por jogo
    melhores jogadores por métrica
    histórico de confrontos
    artilheiros do confronto

Análise com IA

GET /analysis/{team1_id}/{team2_id}

Retorna uma análise simples do confronto com:

    quem está melhor no geral
    vantagens de cada time
    palpite de vencedor

Exemplo da composição do confronto

O frontend agora monta a tela de comparação a partir de múltiplas respostas:

- `GET /teams/{team_id}` para dados completos de cada time
- `GET /matchups/{team1_id}/{team2_id}/history` para histórico de confrontos
- `GET /matchups/{team1_id}/{team2_id}/top-scorers?limit=15` para artilheiros
- `GET /matchups/{team1_id}/{team2_id}/top-players?limit=10` para líderes por métrica
